#画像解説

画面上部に現在時刻と未来の着信時刻を表示。見た目は画像に遵守して、数字はBTTFTimeCircuitsAPDATEDAGAINIMSORRYという名前のフォントを使用して下さい。未入力の数字はーにして下さい。
画面中央のキーパッドの部分はイメージですので背景色に合った色合いで大丈夫です。０の左側の部分は時計のアイコンでお願いします。また画面下部のナビゲーションもお願いします。
背景は似た感じで目に優しい色にして下さい。グラデーションも似た感じでお願いします。

#計画
　キーパッドの各ボタンを押す際に音が鳴るようにします。音は @docs/material/keypadSounds にあるものを使用します。
0 -> @k0.wav
1 -> @k1.wav
2 -> @k2.wav
3 -> @k3.wav
4 -> @k4.wav
5 -> @k5.wav
6 -> @k6.wav
7 -> @k7.wav
8 -> @k8.wav
9 -> @k9.wav
0の右側 -> @kright.wav
0の左側 -> @kleft.wav
コールボタン -> @k0.wav
でお願いします。

バリテーションとして現在時刻と同じ、過去の時刻、上限超過した時刻、13月などの日本時間に存在しない数値が入力され次第DESTINATION TIMEの数字の部分を赤色にして下さい。また全ての部分が入力されそれが問題なければ緑色にして下さい。また赤色、未入力時にコールボタンを押した場合は、録音画面には飛ばず音も @ke.wavが鳴るようにして下さい。現状バリテーションに触れておらず未入力箇所がある場合は白色にして下さい。

現在時刻はリアルタイムで更新されるようにして、変更に合わせてDESTINATION TIMEが現在時刻と同じになってしまったか確認し、同じであれば数字を赤色にしてください。

０の左側のボタンが押されたら下部ナビゲーションバーの裏から出てくる形でキーパッドを隠し数字は隠れない位置に小ウィンドウが現れるようにして下さい、小ウィンドウのイメージ画像はありませんが右上に閉じるボタン、上部にクイックとdatepickerを切り替えるボタンがありそれに合わせてその下にクイックリストかdatepickerが現れます。クイックリスト、datepickerは操作したらその場でDESTINATIONTIMEが変更されるようにして下さい。datepickerの初期値はは開いた時点でDISTINATIONTIMEに数字が入力されていたらそれに合わせて、未入力部分が１つでもあった場合は現在時刻にして下さい。
